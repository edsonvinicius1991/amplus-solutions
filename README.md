# Amplus Solutions

Este repositório contém o código-fonte do projeto Amplus Solutions.

## Configuração do Projeto

Para configurar e executar o projeto localmente, siga os passos abaixo:

### Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas em sua máquina:

- Node.js (versão LTS recomendada)
- npm ou Yarn
- Git

### Instalação

1. Clone o repositório:
   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd amplus-solutions
   ```

2. Instale as dependências:
   ```bash
   npm install
   # ou
   yarn install
   ```

### Execução

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
```

O aplicativo estará disponível em `http://localhost:5173` (ou outra porta, se configurado).

## Requisitos

- **Node.js**: Versão 22.x (LTS) ou superior.
- **Framework**: Next.js com TypeScript.
- **Banco de Dados**: PostgreSQL com Prisma ORM.
- **Cache/Mensageria**: Redis.
